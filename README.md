# Reverie

Reverie is an iOS audio manipulation app built with SwiftUI and AVFoundation. It lets users import audio files, play them back, adjust speed, visualize the waveform, scrub through the track, and export the modified result.

## Features

- Import local audio files
- Play, pause, and stop audio
- Adjust playback speed
- Interactive waveform display
- Seek by tapping through the waveform
- Export modified audio
- Built with a simple, minimal SwiftUI interface

## Tech Stack

- Swift
- SwiftUI
- AVFoundation
- UniformTypeIdentifiers

## Why I Built It

I built Reverie as a personal iOS project to explore audio processing, media playback, waveform interaction, and file export workflows in a native Apple app environment.

## What I Learned

Through this project, I got hands-on experience with:

- iOS app development using SwiftUI
- Audio playback and processing with AVFoundation
- Managing imported files and exports
- Building interactive media UI components
- Synchronizing playback state with waveform tracking

1. Open the project in Xcode
2. Build and run on an iPhone simulator or physical device
